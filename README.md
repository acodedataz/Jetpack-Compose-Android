# 🚀 Jetpack Compose Android (A to Z Fundamentals)

Jetpack Compose Fundamentals from beginner to intermediate level.  
This repository is created for learning modern Android UI development using **Kotlin + Jetpack Compose**.

---

## 📌 About This Project

This project is a step-by-step learning journey of Jetpack Compose, where I practiced:

- Basic UI components (Text, Button, Image, etc.)
- Layouts (Row, Column, Box)
- State management
- Recomposition concept
- LazyColumn & LazyRow (Lists)
- Material 3 UI components
- Navigation in Compose
- UI practice & mini examples

---

## 🛠️ Tech Stack

- Kotlin
- Jetpack Compose
- Android Studio
- Material 3
- MVVM (basic understanding)
